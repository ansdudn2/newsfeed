## 📌주요기능
1.jwt토큰을 이용해 상호작용들이 잘작동되는가

2.팔로우 기능이 잘구현되어있는가

3.여러 다양한 조회기능이 잘 적용되었는가

4.비밀번호를 bcrypt로 인코딩하고있는가

5.데이터베이스에 잘 저장되는가

6.비밀번호를 입력해서 수정과 삭제할때 잘 이용이 되는가.

7.테스트코드가 잘작성되었는가
